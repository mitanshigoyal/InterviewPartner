# AI Interview Practice Partner

Practice mock interviews with AI-powered coaching, voice input, and detailed feedback.

## Features

- Voice Input - Speak your answers naturally
- Smart AI Interviewer - Asks relevant follow-up questions
- Progress Tracking - Monitor your interview in real-time
- Detailed Feedback - Get performance scores and recommendations

## Quick Start

**Install dependencies:**
```bash
pip install streamlit google-generativeai
```

**Get FREE API Key:**
- Visit https://aistudio.google.com/app/apikey
- Sign in and create API key (no credit card needed)

**Run the app:**
```bash
streamlit run .py
```

**Use the app:**
- Enter API key in sidebar
- Select role and experience level
- Start interview and answer questions (type or speak)
- Get feedback after 5-8 questions

## Voice Input

Works with Chrome, Edge, or Safari. Not supported in Firefox.

## Interview Tips

- Use STAR method (Situation, Task, Action, Result)
- Be specific with examples
- Keep answers 2-3 minutes long

## Troubleshooting

**Voice not working?** Check browser compatibility and microphone permissions

**API quota exceeded?** Wait a few minutes or create a new key

**Won't start?** Update packages:
```bash
pip install --upgrade streamlit google-generativeai
```

## Requirements
